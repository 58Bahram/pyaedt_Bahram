Parameters

  • ports -- List of circuit ports for excitation assignation

  • settings -- List of parameters values used in voltage sinusoidal excitation creation:

  • settings[0] -- AC magnitude for small-signal analysis

  • settings[1] -- AC phase for small-signal analysis

  • settings[2] -- DC voltage

  • settings[3] -- Voltage offset from zero

  • settings[4] -- Voltage amplitude

  • settings[5] -- Frequency

  • settings[6] -- Delay to start of sine wave

  • settings[7] -- Damping factor

  • settings[8] -- Phase Delay

  • settings[9] -- Frequency to use for harmonic balance analysis
