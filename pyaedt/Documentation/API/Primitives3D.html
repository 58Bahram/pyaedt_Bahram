Create a Box

>>> hfss = HFSS()
>>> origin = [0,0,0]
>>> dimensions = [10,5,20]

get_face_normal(faceId, bounding_box=None)[source]¶

Get the face normal. Limitations: - the face must be planar. - Currently it works only if the face has at least two vertices. Notable excluded items are circles and ellipses that have only one vertex. - If a bounding box is specified, the normal is orientated outwards with respect to the bounding box. Usually the bounding box refers to a volume where the face lies. If no bounding box is specified, the normal can be inward or outward the volume.

Parameters

  • faceId -- part ID (integer).

  • bounding_box -- optional, bounding box in the form [x_min, y_min, z_min, x_Max, y_Max, z_Max]

Returns

normal versor (normalized [x, y, z]) or None.
